## üöÄ Overview

Devgle is a modern video chat application tailored for developers. It features a unique **"Paper-Craft" aesthetic**, focusing on simplicity and calm. Under the hood, it powers real-time communication using **WebRTC** and decentralized signaling with **GunJS** (or Trystero), ensuring privacy and low latency.

## ‚ú® Features

- **Anonymous Matchmaking**: One-click to find a peer. No login required.
- **HD Video & Audio**: Crystal clear streams via WebRTC.
- **Screen Sharing**: Effortless screen sharing for code reviews and debugging.
- **Paper-Craft UI**: A distinct, soothing design language with "physical" layers, depth, and glassmorphism.
- **Responsive Design**:
  - **Desktop**: Docked Chat Sidebar, Resizable Floating Widgets.
  - **Mobile**: Full-bleed video connection, Swipeable Controls, Mobile-optimized "Zoom-style" experience.
- **Decentralized Chat**: Text chat signaling and storage logic powered by decentralized protocols (GunJS/Trystero).
- **Resizable Widgets**: Customize your workspace by resizing your local camera or screen share preview.

## üì¶ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/GokulAnand14/devgle.git
    cd devgle
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to see the app running.
